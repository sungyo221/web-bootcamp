📑 4주차 정리 — JavaScript 심화

이벤트 · 폼 검증 · 데이터 저장으로 진짜 앱을 만드는 기술.

1. 다양한 이벤트 🖱️⌨️

클릭 말고도 여러 이벤트가 있어요. 패턴은 같아요: 요소.addEventListener("이벤트", 함수)

이벤트언제 발생
"click"클릭할 때
"input"입력창에 글자 칠 때마다 (실시간!)
"keydown"키보드 누를 때 (Enter 감지 등)
"mouseover" / "mouseout"마우스 올릴 때 / 뗄 때

event 객체 — "무슨 일이 일어났나"

input.addEventListener("keydown", function(event) {
  if (event.key === "Enter") {     // 누른 키 확인
    print("엔터!");
  }
});
💡 function(event) 처럼 괄호 안에 event를 받으면, event.key로 어떤 키를 눌렀는지 알 수 있어요.

2. 폼 검증 (Form Validation) ✅

입력값을 조건문으로 검사해서 안내 메시지를 보여줘요. 패턴: 입력 → 검사(if) → 피드백

password.addEventListener("input", function() {
  if (password.value.length < 6) {
    msg.textContent = "6자 이상이어야 해요";
    msg.className = "msg error";   // 빨강
  } else {
    msg.textContent = "안전해요 ✅";
    msg.className = "msg ok";      // 초록
  }
});
코드뜻
.value입력창의 글자
.value.length입력 글자 수
.className = "..."CSS 클래스 교체 (색 제어)
💡 && = "그리고" (둘 다 참이어야 참). || = "또는" (하나만 참이어도 참). 
if (아이디.length >= 4 && 비번.length >= 6) {
  result.textContent = "가입 완료!";   // 둘 다 통과해야!
}

3. localStorage 💾 (데이터 저장)

데이터를 브라우저에 저장해서 새로고침·재방문해도 유지돼요.

코드뜻
localStorage.setItem("키", 값)저장
localStorage.getItem("키")불러오기 (없으면 null)
localStorage.removeItem("키")삭제
// 페이지 열릴 때: 저장된 값 확인
const saved = localStorage.getItem("myName");
if (saved) {
  greeting.textContent = "다시 오셨네요, " + saved + "님!";
}
⚠️ localStorage는 글자(문자열)만 저장돼요! 배열·객체는 변환 필요: 
localStorage.setItem("todos", JSON.stringify(배열));  // 배열 → 글자 (저장)
let 배열 = JSON.parse(localStorage.getItem("todos"));  // 글자 → 배열 (복원)

4. 🧩 데이터 저장 앱의 패턴

To-do 저장 앱에서 배운 핵심 흐름:

[페이지 열림]  getItem → JSON.parse → 배열 복원 → render()
[추가/삭제]    배열 수정 → save() → render()

function save() {
  localStorage.setItem("todos", JSON.stringify(todos));
}
💡 "배열이 바뀌면 무조건 save()." 이 한 줄이 빠지면 새로고침 시 사라져요. 메모앱·가계부·쇼핑몰 전부 이 패턴이에요!

✅ 체크리스트

  • 이벤트는 요소.addEventListener("종류", 함수) — click/input/keydown/mouseover
  • 키 감지는 function(event) + event.key === "Enter"
  • 입력 글자 수는 .value.length
  • "그리고"는 &&, "또는"은 ||
  • 저장은 setItem / 불러오기 getItem / 삭제 removeItem
  • 배열 저장은 JSON.stringify ↔ JSON.parse
  • 요소는 const x = document.getElementById("id") 로 확실히 잡기
