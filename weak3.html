0. JS는 어디에 쓰나?

HTML의 <script> 태그 안에 써요. (CSS는 <style>, JS는 <script>)

<script>
  console.log("안녕");   // F12 콘솔에 출력
</script>

출력 방법: console.log(값) → 개발자도구(F12) 콘솔에 찍힘.

1. 변수 (값을 담는 상자)

let name = "정선교";   // 바뀔 수 있는 값
const age = 26;        // 안 바뀌는 값(상수)
💡 규칙: 일단 const로 쓰고, 값을 바꿔야 할 때만 let. 줄 끝엔 세미콜론 ;

자료형 (값의 종류)

종류예시특징
문자열 string"안녕"★따옴표로 감쌈★
숫자 number42, 3.14따옴표 없음
불리언 booleantrue / false참 / 거짓

typeof 값 → 그 값의 종류를 알려줘요. 글자 잇기는 + : "이름: " + name

2. 연산자

계산비교 (조건에 사용)
+ - * /   %(나머지) >= <= > <   ===(같다) !==(다르다)
⚠️ 가장 중요! "같다"는 === (등호 3개!). = 하나는 "담는다"(변수 대입)예요. 비교는 항상 ===! (== 2개는 버그 위험)

% = 나눈 나머지. 짝수 판별: num % 2 === 0 (나머지 0이면 짝수)

3. 조건문 if / else 🔀

if (점수 >= 90) {
  print("A 등급");
} else if (점수 >= 80) {   // else if 로 단계 추가
  print("B 등급");
} else {                   // 위 조건 다 아니면
  print("재시험");
}

조건이 참(true)이면 그 { } 안을 실행. else if로 여러 단계, else로 나머지 처리.

4. 반복문 for 🔁

for (let i = 1; i <= 5; i++) {
  print(i);     // 1 2 3 4 5
}
부분뜻
let i = 1시작값
i <= 5이 조건이 참인 동안 반복
i++한 바퀴마다 i 를 1 증가
💡 누적 패턴 (자주 씀!): 상자를 0으로 시작해 반복마다 더하기. 
let sum = 0;
for (let i = 1; i <= 100; i++) {
  sum = sum + i;
}
// sum = 5050

5. 함수 function 📦

function 더하기(a, b) {   // 정의 (만들어두기)
  return a + b;          // 결과를 '돌려줌'
}
let 합 = 더하기(7, 5);    // 호출 → 합 = 12
개념뜻
정의함수를 만들어 둠 (아직 실행 X)
호출이름() 으로 실제 실행 (여러 번 재사용)
매개변수함수가 받는 입력 (괄호 안)
return함수가 내놓는 결과 (변수에 담을 수 있음)
💡 print는 화면에 보여주기만, return은 결과를 돌려줘서 다른 곳에 쓸 수 있게 해요.

6. 배열 Array 📚

let 과일 = ["사과", "바나나", "포도"];
과일[0]          // "사과"  (★0부터!★)
과일.length      // 3 (개수)
과일.push("귤")  // 맨 뒤에 추가
과일.splice(1,1) // 1번째 항목 제거
⚠️ 인덱스는 0부터! 첫 번째 = [0], 세 번째 = [2]
💡 배열 + 반복문 = 환상의 짝꿍 (제일 많이 쓰는 패턴!) 
for (let i = 0; i < 과일.length; i++) {
  print(과일[i]);   // 하나씩 전부 꺼내기
}

7. DOM & 이벤트 🖱️ (인터랙티브의 핵심!)

// ① 요소 잡기
const btn = document.getElementById("myBtn");
const box = document.getElementById("myBox");

// ② "클릭되면 이 함수 실행해라"
btn.addEventListener("click", function() {
  box.textContent = "바뀐 글자!";     // 내용 바꾸기
  box.style.background = "#fde68a";    // 스타일 바꾸기
});
코드뜻
document.getElementById("id")id로 HTML 요소 잡기
.textContent = "..."글자 내용 바꾸기
.style.속성 = "..."CSS 스타일 바꾸기
.addEventListener("click", 함수)클릭에 반응
.value입력창(input)의 글자 가져오기

✅ 실수 방지 체크리스트

  • 비교는 === (등호 3개) — =는 대입, ==는 비추천
  • 문자열은 따옴표로 — "글자" / 숫자는 따옴표 없이 42
  • 배열 인덱스는 0부터
  • 줄 끝엔 세미콜론 ;
  • 곱하기는 * (× 아님)
  • 함수는 정의만으론 실행 안 됨 — 호출해야 실행
