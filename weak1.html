📑 1주차 정리 — HTML 기초

웹 페이지의 뼈대(구조)를 만드는 HTML. 1주차에 배운 모든 것을 모았어요.

0. 웹은 어떻게 동작할까?

클라이언트(내 브라우저)가 서버에게 "이 페이지 줘!"라고 요청(request)하면, 서버가 HTML 파일을 응답(response)으로 보내줘요. 이 약속을 HTTP(보안 강화 버전 HTTPS)라고 해요.

언어역할비유
HTML구조 / 뼈대몸의 뼈
CSS디자인 / 꾸미기옷, 화장
JavaScript동작 / 기능움직임, 행동

이번 주는 HTML(뼈대)만 배웠어요. 2주차엔 CSS로 꾸며요!

1. HTML 기본 뼈대

<!DOCTYPE html>
<html lang="ko">
<head>
  <meta charset="UTF-8">      <!-- 한글 안 깨지게 -->
  <title>탭에 뜨는 제목</title>
</head>
<body>
  <!-- 화면에 보이는 내용은 전부 여기 -->
</body>
</html>
  • head = 눈에 안 보이는 정보(제목, 인코딩 등)
  • body = 화면에 보이는 실제 내용
💡 <meta charset="UTF-8"> 를 빼먹으면 한글이 깨져요(����).

2. 자주 쓰는 기본 태그

태그뜻예시
h1 ~ h6제목 (1이 가장 큼)<h1>큰 제목</h1>
p문단(글)<p>내용</p>
a링크<a href="주소">글자</a>
img이미지 (닫는 태그 없음)<img src="주소" alt="설명">
💡 같은 페이지 안 이동: <a href="#menu"> → id="menu" 인 곳으로 점프!

3. 목록과 표 (헷갈렸던 부분! ⚠️)

📋 목록 — ul / li

<ul>
  <li>게임</li>
  <li>운동</li>
</ul>

ul=목록 전체, li=항목 하나. (순서 있으면 ol)

📊 표 — table / tr / th / td

<table>
  <tr><th>메뉴</th><th>가격</th></tr>   <!-- 제목줄: th -->
  <tr><td>아메리카노</td><td>4500원</td></tr>  <!-- 데이터: td -->
</table>
태그역할
tr가로줄 (table row)
th제목 칸 (굵게+가운데, table header)
td데이터 칸 (table data)
⚠️ 섞지 않기! 목록은 li 만, 표는 tr·th·td 만!

4. 폼 (입력 양식)

<form>
  <label for="name">이름</label>
  <input type="text" id="name">

  <label for="review">후기</label>
  <textarea id="review"></textarea>

  <button type="submit">보내기</button>
</form>
태그역할
label입력칸 설명 글자
input한 줄 입력 (type으로 종류 결정)
textarea여러 줄 입력
button버튼 (type="submit"=제출)

input type 종류: text · email · password · number · date · checkbox

⚠️ 핵심! label의 for = input의 id 를 글자 하나까지 똑같이 맞춰야 짝이 돼요. (예: for="name" ↔ id="name")

5. 시맨틱 태그 (페이지 구조)

그냥 div 대신 의미 있는 이름의 상자로 페이지를 나눠요.

<header>     머리말 (제목 h1 + 메뉴 nav)
<nav>        메뉴 (보통 ul+li+a)
<main>       본문 핵심 (★페이지에 1개만!★)
  <section>  주제별 구역
<footer>     꼬리말 (저작권 등)
태그위치/역할
header맨 위 머리말 — h1·nav를 안에 감싸기
nav메뉴 모음
main본문 (한 페이지 1개)
section주제별 구역
footer맨 아래 꼬리말
⚠️ 자주 한 실수 2가지
1) h1·nav는 반드시 <header>...</header> 안에 넣기
2) footer는 </main> 바깥(뒤)에 두기

6. 전체 골격 한눈에 보기

<body>
  <header>
    <h1>제목</h1>
    <nav><ul><li><a href="#menu">메뉴</a></li></ul></nav>
  </header>

  <main>
    <section id="menu"> ... </section>
    <form> ... </form>
  </main>

  <footer><p>© 2026 정선교</p></footer>
</body>
