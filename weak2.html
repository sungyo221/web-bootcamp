📑 2주차 정리 — CSS

웹 페이지를 꾸미는(디자인) CSS. 2주차에 배운 모든 것을 모았어요.

0. CSS는 무엇?

HTML = 뼈대, CSS = 옷·화장. 같은 HTML도 CSS에 따라 완전히 달라 보여요.

기본 문법 (딱 이 모양!)

선택자 {
  속성: 값;     /* ← 끝에 세미콜론 ; 꼭! */
}

/* 예시 */
h1 {
  color: blue;
  font-size: 30px;   /* ← 숫자엔 단위 px 꼭! */
}

"어떤 걸(선택자) → 뭐를(속성) → 어떻게(값)" 바꿀지 정하는 거예요.

1. CSS를 HTML에 넣는 3가지 방법

방법형태비고
인라인<h1 style="color:red">지저분, 비추천
내부<style>...</style> (head 안)연습용
외부<link rel="stylesheet" href="style.css">실무 정석!
💡 외부 CSS: 구조(HTML)와 디자인(CSS)을 다른 파일로 분리. <head> 안에 <link> 한 줄로 연결해요. (우리가 cafe.html ↔ cafe.css 로 해본 방식!)

2. 선택자 3종 (제일 중요!)

선택자고르는 대상예시
태그그 태그 전부p { } → 모든 <p>
class (점 .)class="..." 붙은 것.box { } → class="box"
id (샵 #)id="..." 붙은 것(1개)#title { } → id="title"

여러 개 한꺼번에: th, td { } (쉼표로 묶기) · 자식만: nav a { } (nav 안의 a)

⚠️ class는 점 ., id는 샵 #. CSS에서 붙이는 걸 잊지 마세요! (HTML 태그엔 class="...", id="..." 그대로)

3. 자주 쓰는 속성

속성뜻예시
color글자색color: white;
background-color배경색background-color: #6366f1;
font-size글자 크기font-size: 22px;
font-weight글자 굵기font-weight: bold;
text-align가로 정렬text-align: center;
text-decoration밑줄 등text-decoration: none; (링크 밑줄 제거)
list-style목록 점list-style: none; (점 제거)
border-radius모서리 둥글게border-radius: 10px;
cursor마우스 모양cursor: pointer; (손가락)
⚠️ 색 표기: red, white 같은 이름 또는 #6366f1 같은 6자리 코드(#RRGGBB).

4. 박스 모델 📦 (가장 핵심!)

모든 요소는 네모 상자. 4겹으로 되어 있어요.

margin (바깥 여백 = 노란 영역)
border (테두리) + padding (안쪽 여백 = 파란 영역)
↑ 내용(content)
층뜻특징
content실제 내용글·이미지
padding안쪽 여백 (내용~테두리)배경색이 칠해짐
border테두리 선border: 2px solid #333; (두께 종류 색)
margin바깥 여백 (다른 상자와 거리)투명 (안 칠해짐)
💡 방향별: padding: 10px 20px; (위아래 10 / 좌우 20) · margin-top: 16px; (위만)
⚠️ padding vs margin — padding은 상자 안쪽(배경색 칠해짐), margin은 상자 바깥(투명, 다른 요소 밀어냄).

5. Flexbox ↔️ (가로 배치의 왕)

요소를 가로로 나란히 배치하고 정렬. 실무에서 제일 많이 써요.

.container {
  display: flex;     /* ← 이 한 줄이면 자식들이 가로로! */
}
속성역할자주 쓰는 값
display: flex가로 배치 시작(★flexbox 아님! flex★)
justify-content가로 정렬center / space-between
align-items세로 정렬center
gap자식 사이 간격gap: 16px;
💡 메뉴바 공식: display:flex + justify-content: space-between(로고 왼쪽·메뉴 오른쪽) + align-items: center(세로 가운데).
⚠️ 가장 흔한 실수! display: flexbox; ❌ → display: flex; ✅ ("Flexbox"는 기술 이름, CSS 값은 flex)

6. 색 대비 (디자인 기본기)

✅ 어두운 배경 + 흰 글자 → 잘 보임

❌ 밝은 배경 + 흰 글자 → 흐릿해서 잘 안 보임

💡 규칙: 흰 글자 → 어두운 배경 / 어두운 글자 → 밝은 배경. 글자와 배경은 항상 밝기 차이를 크게!
