🗺️ 12주 학습 로드맵

매주 학습 항목을 체크하면 위쪽 진행률이 자동으로 올라가고, 브라우저에 저장돼요.

1 1단계 · 웹의 기초 & 프론트엔드 입문 (1~3주) 0/0

1주차 — 웹 동작 원리 & HTML

목표: 웹이 어떻게 동작하는지 이해하고, HTML로 문서 구조를 만든다.

📌 미니 프로젝트: HTML만으로 "자기소개 페이지" 만들기

2주차 — CSS & 반응형

목표: CSS로 디자인하고, 어떤 화면에서도 보기 좋게 만든다.

📌 미니 프로젝트: 반응형 "랜딩 페이지"(소개 사이트) 만들기

3주차 — JavaScript 기초

목표: 프로그래밍의 기본 문법과 사고방식을 익힌다.

📌 미니 프로젝트: 계산기 또는 가위바위보 게임(콘솔/간단 UI)
2 2단계 · JavaScript 심화 & 동적 웹 (4~6주) 0/0

4주차 — DOM & 이벤트

목표: JavaScript로 화면을 직접 조작하고 사용자와 상호작용한다.

📌 미니 프로젝트: To-do 리스트 앱 (추가/삭제/완료/저장)

5주차 — ES6+ & 비동기/API

목표: 현대 JS 문법과 외부 데이터(API)를 다룬다.

📌 미니 프로젝트: 공개 API로 날씨 앱 / 영화 검색 앱

6주차 — Git, 협업도구 & 배포

목표: 실무 필수 도구를 익히고 내 결과물을 인터넷에 올린다.

📌 목표: 배포된 프로젝트 링크 2~3개 확보 (포트폴리오 시작!)
3 3단계 · 프레임워크 & 백엔드 입문 (7~9주) 0/0

7주차 — React 기초

목표: 컴포넌트 기반 프론트엔드 개발을 시작한다.

📌 미니 프로젝트: React로 To-do 앱 다시 만들기

8주차 — React 심화 & SPA

목표: 여러 페이지와 API를 다루는 실제 앱 구조를 만든다.

📌 미니 프로젝트: API 기반 SPA(영화/상품 목록 + 상세 페이지)

9주차 — Node.js, Express & DB

목표: 직접 서버와 API, 데이터베이스를 만든다.

📌 미니 프로젝트: React 프론트 + Express API + DB 연동(메모/게시판)
4 4단계 · 풀스택 통합 & 취업 준비 (10~12주) 0/0

10~11주차 — 풀스택 포트폴리오 프로젝트

목표: 취업용 "대표 프로젝트" 1개를 제대로 완성한다.

📌 산출물: 실제 접속 가능한 풀스택 앱 + 잘 정리된 GitHub

12주차 — 취업 준비 마무리

목표: 이력서·포트폴리오·면접 기초를 갖추고 지원을 시작한다.

🎯 최종 목표: 배포된 풀스택 앱 + GitHub 포트폴리오 + 이력서 + 지원 시작
